# 📚 Sistema de Gerenciamento de Livros

Este projeto é uma API REST construída com **Spring Boot**, utilizando **JPA/Hibernate** para persistência de dados em um banco **MySQL**. A aplicação gerencia livros, autores e categorias com operações CRUD completas.

---

## ⚙️ Tecnologias Utilizadas

- Java 21
- Spring Boot
- Spring Data JPA
- MySQL
- Lombok
- Maven

---

## 🏗️ Estrutura da Aplicação

- `model`: Entidades JPA (Livro, Autor, Categoria)
- `dto`: Objetos de transferência de dados
- `repository`: Interfaces para comunicação com o banco
- `service`: Lógica de negócio e validações
- `controller`: Endpoints REST para operações CRUD
